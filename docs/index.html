cyclical 태그는 분류용 라벨일 뿐 매수/매도 신호는 없음(가격→팩터 역방향 재검증 결과 반도체 외엔 타이밍 신호로 못 씀 — 결론 확정, 신호는 대시보드에서 제거). 개별 업종 국면 판단은 참고 문서로: 종합분류 · 반도체 · 에너지·소재 · 건설 · 레저·리테일·자본재·항공방산·운송

이익 추세 판단 — 영업이익 CAGR 기준
라벨조건 (연환산 CAGR)
▼▼ 2y역성장1y CAGR < 0 and 2y CAGR < 0 — 2년 연속 영업이익 감소
▼ 1y역성장1y CAGR < 0 — 최근 1년 영업이익 감소
▲▲ 가속1y CAGR > 0 and 1y CAGR > 2y CAGR — 성장 중이고 속도가 빨라지는 중
▲ 성장1y CAGR > 0 — 영업이익 증가 (속도는 둔화)
— 보합기타 (데이터 부족 등)

CAGR 계산: TTM(직전 4분기 누적) 기준, 분기 단위 기하평균 → 연환산. 1y = 최근 4분기 vs 직전 4분기.

밸류 평가 판단 — 현재 배수 vs 4년 역사 평균
라벨조건
저평가현재(최근 20일 평균) < 4년 평균 × 0.75 — 역사 대비 25% 이상 할인 (백테스트 최적 임계값)
고평가현재(최근 20일 평균) > 4년 평균 × 1.25 — 역사 대비 25% 이상 프리미엄
혼재배수별로 방향이 엇갈림 — ↓ 표기된 배수가 현재 저평가 쪽
중립모든 배수가 ±25% 이내

판정 대상 배수: P/E · P/S · P/OP · P/FCF 각각 독립 판정. 예) "저평가 (P/S+P/E)" = P/S와 P/E 현재값이 4년 평균보다 25%+ 낮음. 4년 평균은 과거 16분기 기하평균.

Growth·Value 공통 방법론: PIT(point-in-time) 보정 — 16분기 연속 흑자를 각 시점마다 롤링 재판정한 유니버스(2013Q4~2026) 사용. alpha는 평균이 아닌 중앙값(median), 전부 SPY(S&P 500) 대비 초과수익. 방법론 상세는 STATUS.md 참조.
