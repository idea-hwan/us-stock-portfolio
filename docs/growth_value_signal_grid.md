# Growth/Value 매수 신호 재검증 (2026-07-22)

## 결론

현재 프로덕션 신호 — `rev2y가속 + P/OP저평가 + capex_1y↓` — 를 바꿀 근거 없음.
PIT 교정 데이터(`pit_buckets16`, 생존편향 제거 + 401종목 확장 유니버스)로 3축(성장·저평가·투자) 팩터를 전수조합해도 growth·value 두 버킷 모두에서 **1위**.

- growth: n=265, 12m +15.0%p, 18m +20.0%p (베이스 대비, 원값 12m +16.3%/18m +22.0%)
- value: n=216, 12m +19.8%p, 18m +25.7%p (베이스 대비, 원값 12m +21.2%/18m +27.8%)

`signal_value()`가 `signal_growth()`와 같은 공식을 쓰는 것도 근거 있음 — value 버킷 자체 그리드에서도 같은 조합이 1위.

과거(PIT 이전) 그리드는 반대 방향("역성장"+"P/OP저평가는 역효과")을 최선으로 봤는데, 그건 생존편향 낀 옛 데이터의 아티팩트였던 것으로 보임 — PIT 교정 후 순위가 뒤집힘.

**주의**: 1위 조합도 표본이 전체의 1.8~2.1%뿐(n=216~265). 784개 조합을 스캔해 상위를 뽑는 구조라 다중비교 문제에서 자유롭지 않음. "확정"이 아니라 "PIT 데이터에서도 살아남은 신호" 정도로 취급.
