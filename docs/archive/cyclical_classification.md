## 분류 원칙

### 수치 기준 (선정 근거)

경기순환주의 본질은 **영업이익의 zero-crossing** — 경기 수축기에 영업이익이 적자로 전환되거나, 적자 직전까지 급락하는 패턴.

수동 리스트(`cyclical_universe.txt`)는 아래 지표를 기준으로 선별했다:

| 기준 | 내용 | 관련 컬럼 |
|------|------|----------|
| **영업이익 zero-crossing** | 시계열 구간 내 영업이익(`op_ps`)이 음수를 경험 | `op_geom_*_q_minus > 0` |
| **Capex 변동성** | Capex가 경기와 함께 크게 출렁 (자본집약적 업종) | `capex_geom_*_mcum` 변동폭 |
| **FCF 급감** | Capex 급증 시 FCF 동반 급락 | `fcf_geom_*_mcum` |
| **업종 사이클** | 업종 고유의 수요 사이클 존재 확인 | 수동 판단 |

> zero-crossing이 없더라도 Capex 집중도가 높고 FCF가 경기와 강하게 연동되면 분류.  
> zero-crossing이 있어도 규제 산업(방산·유틸리티) 또는 구조적 업황 하락이면 제외.

## 진입 팩터 분석

세 축으로 검증: **① 돈 얼마나 벌었나** (매출·이익·현금 성장률) / **② 돈 얼마나 썼나** (capex 방향) / **③ 주가 싸냐** (배수 vs 4y 평균)

### 팩터 설계 원칙

**밸류에이션 신호** — 단일 배수보다 복수 배수 동시 저평가가 훨씬 강함.  
배수별 특성:
- `ps` (P/매출): 매출은 이익보다 안정적이고 사이클 초입에 먼저 회복 → 대부분 업종 핵심
- `pfcf` (P/FCF): 자산집약 업종(설비·주택·항공우주)에서 FCF가 사이클과 강하게 연동
- `pop` (P/OP): 제조 경기순환에 유효하나 **retail·semiconductor에서 역효과**
- `pe` (P/E): 단독으로는 약하나 조합 강화에 기여

**capex 신호** — 업종 성격에 따라 방향이 반대:
- 반도체: capex = 공급측 장비 발주 → 삭감이 공급 감소 예고 (매수)
- 기타 경기순환: capex = 수요 반응 사업 확장 → 확대가 사이클 회복 신호 (매수)

**이익/매출 신호** — 성장·방향·기간(1y/2y) 조합

### 업종별 팩터 해석

**construction**: 착공·수주 사이클이 명확해 밸류에이션 저평가만으로 강력한 신호. ps(P/매출)가 핵심 — 착공 감소로 매출이 눌릴 때 주가가 더 과도하게 빠진다.

**aerospace_defense**: 항공기 납품 백로그(8~10년) 사이클. pop+ps+pe 동시 저평가 = 시장이 납품 지연을 과도하게 할인한 구간.

**housing**: 금리 민감 업종. ps+pe+pfcf — 매출(착공수)·순이익·FCF 세 방향에서 다 싸야 진짜 바닥. pfcf가 중요한 이유: 주택건설사의 토지 매입 capex가 사이클과 직결.

**capital_goods**: val + **capex 삭감 2y**가 최강. 설비기업이 자사 투자를 2년째 줄이는 구간 = 사이클 깊은 저점. 주가까지 저평가 = 바닥 확인 신호.

**leisure**: val + capex 확대 + op 회복. 호텔·여행은 투자(신규 객실·플랫폼)를 늘릴 때 수요 자신감 신호. capex 삭감 구간은 역효과.

**retail**: pop이 역효과(소매 이익 저점 = 구조 약화 가능). pop+pe+pfcf + capex 확대 — 재고 투자 재개가 수요 회복 신호.

**semiconductor** *(별도 로직)*: 밸류에이션 신호는 부차적(ps+pfcf +4.3%p). 핵심은 이익 사이클 — 2년 연속 영업이익 역성장이 재고 사이클 바닥 신호.

### 반도체 전용 팩터 해석

**왜 일반 밸류에이션이 역효과인가**  
P/OP가 낮을 때(이익 고점) = 사이클 정점, P/OP가 높을 때(이익 저점) = 사이클 바닥.  
일반 경기순환주와 밸류에이션 신호가 역전된다. capex도 반대 — 장비 발주 삭감 = 공급 감소 예고 = 매수.

**capex 방향도 반대인 이유**  
기타 경기순환: capex = 수요 반응 (확대가 매수 신호)  
반도체: capex = 공급측 장비 발주 (삭감이 공급 감소 예고 → 매수 신호)

#### 핵심 발견

**1. 이익이 감소 중일 때 사라 (반도체 역설)**  
이익 회복(②)이 확인되면 이미 선반영. 이익 감소(③)가 확인되는 시점이 진짜 매수 구간.

**2. op 2년 연속 감소 = 가장 강한 단일 팩터** (`op_geom_2y_mcum < 0`, 12m +12.9%, 18m +21.0%)  
단순 단기 하락이 아니라 깊은 저점 확인. 반도체 재고 사이클 바닥을 정밀 포착.

**3. capex 삭감 = 공급 감소 예고 신호** (⑧ 12m +5.4%)  
메모리 제조사·장비사가 capex를 줄이면 다음 사이클 공급 부족으로 이어짐.

**4. 삼중 바닥(⑬)은 18m에서 최강** (18m +20.4%)  
`이익↓ + capex↓ + pop 비싸 보임` — n=107로 제한적이지만 신뢰도 높은 바닥 시그널.

**5. 이익 회복 + pop 비싸 보임(⑯) = 사이클 정점 신호** (12m −4.9%)  
이미 주가 상승으로 P/OP가 낮아 보이는 상황에서 이익도 회복 중이면 = 출구 신호.

**6. 보유 기간은 18m가 맞음**  
12m → 18m에서 알파가 약 2배. 반도체 사이클 한 바퀴가 12개월보다 길다.

## 향후 검토 사항

- **transport**: ODFL이 혼자 +15.2%, 나머지 전부 음수. 섹터 단위 알파보다 ODFL 단독 분석이 유효할 수 있음

- **leisure**: ABNB는 데이터 축적 후 재평가 필요
- **semiconductor**: 진입 팩터 분석 완료 → `op_geom_2y_mcum < 0` (18m +21.0%). 보유 기간 18m 권장.
