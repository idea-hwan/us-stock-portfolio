### 핵심 발견

**1. 역성장 구간이 최강 진입 시점**  
성장주는 이익이 일시적으로 눌릴 때 시장이 과도하게 할인한다.  
`op_geom_1y_mcum < 0` 단독으로 베이스 대비 +1.7%p.

**2. 최강 신호: 영업이익 눌림 + 매출 가속**  
`op_geom_1y_mcum < 0 AND rev_geom_1y_mcum > rev_geom_2y_mcum`  
→ 매출은 오히려 빨라지는데 영업이익만 역성장 = 투자·비용 선집행 구간.  
구조는 건전한데 시장이 이익 숫자만 보고 과매도.

**3. 2년 연속 영업 역성장 + 매출 가속이 12m 최강 (n=228, +7.7%)**  
`op_geom_1y_mcum < 0 AND op_geom_2y_mcum < 0 AND rev_geom_1y_mcum > rev_geom_2y_mcum`

**4. 18m 최강: op 2y역성장 + 매출 2y 가속 (n=299, +10.6%)**  
`op_geom_2y_mcum < 0 AND rev_geom_2y_mcum > rev_geom_4y_mcum`

**5. P/OP, P/E 저평가 = 역효과**  
경기순환주와 달리 성장주에서 이익 배수 저평가는 구조 약화 신호일 수 있음.  
P/S는 약하게 긍정 — 성장주는 매출 배수가 더 적절한 밸류에이션 기준.

**6. 역설 — 영업역성장 + 순이익정상 = 최악 (−2.7%)**  
비영업이익으로 순이익을 유지 중인 경우. 영업 구조 약화가 숫자에 가려진 상태.

### 권장 진입 팩터

```
# 핵심 (12m 최적)
op_geom_1y_mcum < 0
AND op_geom_2y_mcum < 0

# 보수적 (n 확보)
op_geom_1y_mcum < 0

# 장기 보유 (18m 최적)
op_geom_2y_mcum < 0
AND rev_geom_2y_mcum > rev_geom_4y_mcum   # n=299, 18m +10.6%
```

### 핵심 발견

**가치주에서는 진입 타이밍 팩터가 작동하지 않는다.**

성장주·경기순환주와 달리 가치주는 이익 역성장이나 밸류에이션 저평가가 매수 신호가 아니라 구조 약화 신호다. 역성장 + 저평가 조합은 오히려 알파를 대폭 악화시킨다.

**가치주 팩터의 역할: 제외 필터 (exclusion filter)**

| 제외 조건 | 의미 |
|----------|------|
| 매출 2년 연속 역성장 | 구조적 매출 쇠퇴 — 사이클이 아닌 시장 축소 |
| P/S 저평가 + 영업역성장 | 매출 대비 싼 이유가 있음 — 이익이 구조적으로 압축 중 |
| P/OP 저평가 + 영업역성장 | "싸 보이지만 더 싸질" 구간 — 이익 하락 중 배수 저평가 |

### 권장 사용법

```
# 가치주는 진입 타이밍이 아닌 제외 필터로 활용

# 투자 회피 조건 (이 조건 해당 시 skip)
rev_geom_2y_mcum < 0                              # 매출 2y 역성장
OR (ps_20d < ps_4y AND op_geom_1y_mcum < 0)      # P/S 저평가 + 영업 역성장
OR (pop_20d < pop_4y AND op_geom_1y_mcum < 0)    # P/OP 저평가 + 영업 역성장

# 긍정 신호 (약하지만 유일하게 작동): 영업역성장 + 매출가속
# n이 너무 작아 실용적 사용은 어려움 (+0.9%p, n~450)
```

### 해석

가치주는 이미 성장이 둔화된 기업들의 집합이다. 여기서 이익이 더 나빠지면 회복 기대가 아닌 구조 악화다. 경기순환주처럼 "싸 보이면 사라"가 통하지 않는다.

유효한 투자 방식: **개별 종목 선별** (사업 경제성·경쟁 우위 분석) + 베이스라인 조건(매출 역성장 없음, 이익 배수 고평가 아님) 정도만 스크리닝.
